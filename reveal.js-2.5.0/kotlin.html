Kotlin概要


fun main(args: Array<String>): Unit { 
    println("Hello world!") 
}

JetBrains社作 (その他IntelliJ IDEA Java IDEを開発)

2011年開発開始

Apache Public Licence 2

現在開発中

JVM用とJavascript用コンパイラーあり(.NET検討中)

基本機能がJavaと共通

  • 静的型付き
  • オブジェクト指向
  • ジェネリックス対応
  • 例外 ... など

はやりの無名関数や高階関数に対応(Scala, Java 8に同じ) 


array(-2, -1, 0, 1, 2).filter {it > 0} // -> (1, 2)

Kotlin特徴

静的なNull Pointer Exception抑制 


var a : String = "abc" 
a = null // エラー!
nullの可能性は明示的に指定 

var b : String? = "abc" 
b = null // OK
b.length() // エラー: nullの可能性
nullチェックは必須 

val length : Int = if (b!=null) b.length() else -1
//または
val length : Int? = b?.length() // OK

NPEを起こしたい人は b!!.length()でどうぞ

Primitive型廃止

動的に最適化をしてくれる: 


val i : Int = 1 // -> int
val i : Int? = 1 // -> Integer

パターンマッチングとスマートキャスト 


val x : Any = ...
when (x) { 
  is Int -> print(x) 
  is List<*> -> print(x.size()) // スマートキャスト!
  !is Number -> print("Not even a number") 
  else -> print("can't do anything") 
}

その他、Scala/Javaにない機能

  • 静的拡張関数 (例: 123.isZero())
  • inline関数で高階関数のオーバヘッド削減
  • 移譲(delegation)パターンを言語レベルで対応
